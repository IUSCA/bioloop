| **readonly** | Makes switch read only | `Boolean` | `false` |
| **rules** | Validation rules . | `Array` | `[]` |
| **size** | Specify size for component. `"small"`, `"medium"`, `"large"` sizes are available | `String` | `"medium"` |
| **stateful** | Add possibility to work with component without setting `v-model`. | `Boolean` | `false` |
| **success** | Show component in success state. | `Boolean` | `false` |
| **trueInnerLabel** | Inner label when checked | `String` | `null` |
| **trueLabel** | Label when checked | `String` | `null` |
| **trueValue** | Value when checked | `any` | `true` |
